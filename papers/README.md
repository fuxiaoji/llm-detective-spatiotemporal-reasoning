# Papers

This folder stores papers related to the project idea:

> Test-time reasoning and structured spatio-temporal evidence memory for LLM detective reasoning.

## CoT / Test-Time Improvement

- `2305.10601_Tree_of_Thoughts.pdf`  
  Tree of Thoughts. Baseline for multi-path deliberate reasoning.

- `2025.emnlp-main.1025_s1_Simple_Test_Time_Scaling.pdf`  
  s1. Simple test-time scaling and budget forcing.

- `2605.17609_Adaptive_Generate_Rank_Verify.pdf`  
  Adaptive generate-rank-verify. Cost-aware inference-time search with costly verification.

- `2310.16049_MuSR_Multistep_Soft_Reasoning.pdf`  
  MuSR. Includes a murder mystery setting for multi-step soft reasoning.

- `2406.12641_DetectBench.pdf`  
  DetectBench. Implicit evidence discovery and multi-hop detective reasoning.

- `2409.02465_DetectiveQA.pdf`
